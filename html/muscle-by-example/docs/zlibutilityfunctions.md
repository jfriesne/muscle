# muscle::ZLibUtilityFunctions class [(API)](https://public.msli.com/lcs/muscle/html/group__zipfileutilityfunctions.html)

```#include "zlib/ZLibUtilityFunctions.h"```

The functions in [ZLibUtilityFunctions](https://public.msli.com/lcs/muscle/html/group__zlibutilityfunctions.html) are convenience functions for quickly deflating or inflating data.

* [DeflateByteBuffer()](https://public.msli.com/lcs/muscle/html/group__zlibutilityfunctions.html#gac0e7111c0ac02972add4180fdda46bff) takes a [ByteBufferRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1ByteBuffer.html) of raw data and returns a corresponding [ByteBufferRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1ByteBuffer.html) of deflated data
* [InflateByteBuffer()](https://public.msli.com/lcs/muscle/html/group__zlibutilityfunctions.html#ga0e55722593dbcd0d8b9c68c8f89b2ffb) takes a [ByteBufferRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1ByteBuffer.html) of deflated data (tht was previously returned by [DeflateByteBuffer()](https://public.msli.com/lcs/muscle/html/group__zlibutilityfunctions.html#gac0e7111c0ac02972add4180fdda46bff)) and returns a corresponding [ByteBufferRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1ByteBuffer.html) of re-inflated data
* [DeflateMessage()](https://public.msli.com/lcs/muscle/html/group__zlibutilityfunctions.html#ga79ec62422327c90210d63c4ccea9beda) takes a [MessageRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html) and returns a corresponding [MessageRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html) containing only a single field of flattened data (but with the same [what](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html#afaa29abc90165cc887596fd8021e9795)-code as the original [Message](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html)) -- or if the [MessageRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html) that was passed in was already deflated, it just returns the passed-in [MessageRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html) verbatim.
* [InflateMessage()](https://public.msli.com/lcs/muscle/html/group__zlibutilityfunctions.html#ga1f80cc7fc134596989b9e4084e06b17e) takes a [MessageRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html) that was previously returned by [DeflateMessage()](https://public.msli.com/lcs/muscle/html/group__zlibutilityfunctions.html#ga79ec62422327c90210d63c4ccea9beda) and returns a corresponding re-inflated [MessageRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html) containing all of the original fields again.  (or if the [MessageRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html) that was passed in wasn't a deflated Message, it just returns the passed-in [MessageRef](https://public.msli.com/lcs/muscle/html/classmuscle_1_1Message.html) verbatim)
* [ReadAndDeflateAndWrite()](https://public.msli.com/lcs/muscle/html/group__zlibutilityfunctions.html#ga768e986b8a82b6009f28a930bbb0bbd6) reads raw data from one [DataIO](https://public.msli.com/lcs/muscle/html/classmuscle_1_1DataIO.html), deflates it, and writes the deflated data out to another [DataIO](https://public.msli.com/lcs/muscle/html/classmuscle_1_1DataIO.html), without ever loading all of the data into RAM at once.  (Useful for deflating large files)
* [ReadAndInflateAndWrite()](https://public.msli.com/lcs/muscle/html/group__zlibutilityfunctions.html#ga565f2b388d1249b7322ade31a6388760) reads deflated data from one [DataIO](https://public.msli.com/lcs/muscle/html/classmuscle_1_1DataIO.html), inflates it, and writes the inflated data out to another [DataIO](https://public.msli.com/lcs/muscle/html/classmuscle_1_1DataIO.html), without ever loading all of the data into RAM at once.  (Useful for inflating large files)
