muscle::UDPSocketDataIO class (API)

#include "dataio/UDPSocketDataIO.h"

UDPSocketDataIO handles transmitting and receiving data via UDP packets.

  • UDPSocketDataIO is a subclass of PacketDataIO, so it has the additional packet-I/O related functionality described in that class.
  • SetPacketSendDestination(const IPAddressAndPort &) can be called to specify where PacketDataIO::Write() should send its outgoing UDP packets should be sent to
  • SetPacketSendDestinations(const Queue<IPAddressAndPort> &) can be called to specify that PacketDataIO::Write() should send each outgoing UDP packet to multiple destinations.

Try compiling and running the mini-example-programs in muscle/html/muscle-by-example/examples/dataio (enter make to compile example_*, and then run each from Terminal while looking at the corresponding .cpp file)
